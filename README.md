pinyin4j
========

A copy of http://sourceforge.net/projects/pinyin4j, then deploy it to maven central repository.

### Download ###
Download the jar via maven:
```xml
<dependency>
    <groupId>com.belerweb</groupId>
    <artifactId>pinyin4j</artifactId>
    <version>2.5.0</version>
</dependency>
```
